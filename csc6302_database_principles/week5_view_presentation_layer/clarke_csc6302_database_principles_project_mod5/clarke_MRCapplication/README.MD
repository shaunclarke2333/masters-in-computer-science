# How to Run the MRC Application

Follow the steps below to install dependencies and start the application.

## Navigate to the Project Directory
From your terminal, move into the project folder:


- cd clarke_MRCapplication
- Install Required Packages
- The application includes a requirements.txt file that lists all necessary Python dependencies.

  - Install them using:
  - pip install -r requirements.txt
    - (Use pip3 if your system requires it.)

## Run the Application
Start the Flask app by running:
- python3 app.py
  - (Or python app.py depending on your Python installation.)

## Access the Application in Your Browser
After running the app, the Flask server will display an address similar to:

- Running on http://127.0.0.1:5000/
- Open that URL in your web browser to access the MRC application.
